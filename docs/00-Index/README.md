# 00-Index — Maps of Content (MOCs)

โฟลเดอร์นี้รวม MOCs และ status doc — ใช้เป็น entry point เข้าทุก vault

## ไฟล์หลัก

- [[IMPLEMENTATION-STATUS]] — สถานะ project + features ทุก phase (single source of truth)
- [[MOC-PRD]] — รายการ PRD ทั้งหมด
- [[MOC-Features]] — รายการ features
- [[MOC-Functions]] — รายการ function specs

## รูปแบบ MOC

แต่ละ MOC เป็น list ที่ link ไปไฟล์จริง + status ปัจจุบัน — update โดย /bda-doc และ /bda-implement

## สำหรับ AI

ก่อนเริ่มงานใดๆ อ่าน `IMPLEMENTATION-STATUS.md` ก่อนเสมอ
