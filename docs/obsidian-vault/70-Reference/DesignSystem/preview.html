Accessibility checklist

  • Contrast ratio ≥ 4.5:1 สำหรับ body, ≥ 3:1 สำหรับ large text
  • Focus ring 2px solid primary-500 + offset 2px ทุก interactive element
  • Disabled state มี opacity 0.5 + cursor not-allowed (ไม่ใช่สีอ่อนเฉยๆ)
  • Form ทุกตัวมี <label> หรือ aria-label
  • Modal มี focus trap + ปิดด้วย ESC
  • Color ห้ามเป็น sole indicator (เช่น error ต้องมีไอคอนหรือ text ร่วม)
  • Loading state มี aria-busy หรือ role="status"
