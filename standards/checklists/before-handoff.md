# Checklist — Before Handoff

ก่อน `/bda-verify` → handoff:

- [ ] Plan/fix file status: done
- [ ] All test scenarios PASS หรือ acknowledged BLOCKED
- [ ] Evidence stored ใน `docs/90-TestPlan/evidence/<slug>/`
- [ ] Manifest.json มี + safe_to_share validated
- [ ] PII masking confirmed
- [ ] Vault link graph ครบ ([[…]] ทุกอันชี้ไปไฟล์จริง)
- [ ] `00-Index/IMPLEMENTATION-STATUS.md` updated
- [ ] Design system compliance: 0 violations
- [ ] Security pre-flight: ALL GREEN หรือ justified
- [ ] Handoff doc ใน `docs/95-Handoff/HOR-<slug>.md` มี 5 mandatory sections ครบ
- [ ] Rollback / mitigation plan (ถ้า production-facing)
- [ ] Reviewer assigned + notified
